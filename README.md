## 琴谱制作

采用的方式是，直接抄写数字

（1）高音在前面加点，低音在后面加点

比如：
     .
|1 2 3 4|
 . .      

这一句的数组写法
String arr[] = {"1.","2.",".3",".4"};

(2)连音  

  -
|1 2 3 4|

(3)空拍

空拍使用0

|1 2 3 -|

String arr[] = {1,2,3,0};

(4)长音使用+

|1.  2 3 4|

String []arr = {1+,2,3,4};

(5)短音使用减

|1 2 3 4|
 _

    -1,2,3,4
};
